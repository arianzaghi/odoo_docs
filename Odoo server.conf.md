## Pycharm Run/Debug Configurations

- **script:** /opt/sources/odooXXX/src/core/odoo-bin

- Config file
```
-c path/to/client.conf
```
- Database
```
-d DATABASE_NAME	
```
- Update module
```
-u MODULE_NAME
```
- Force update translations

```
--i18n-overwrite
```
