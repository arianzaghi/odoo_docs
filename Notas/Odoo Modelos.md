# Odoo Models

## Nomenclaturas

- **Ficheros**
	Aplica tanto para vistas como clases
	- Ficheros de herencia de clases existentes
		`sale_order.py`
	- Ficheros de clases nuevas
		`pnt_mi_clase_nueva.py`

- Clases
	- Clases nuevas
		`PntClaseNueva(Models.model)`
	- Herencia de clases
		`SaleOrder(Models.model)`

- **Variables**
	`pnt_variable_nueva`
