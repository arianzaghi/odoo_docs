# botones colores

### Button Classes:

Here are some common Bootstrap classes you can use for different button colors:

- **`btn-primary`**: Blue
- **`btn-secondary`**: Gray
- **`btn-success`**: Green
- **`btn-danger`**: Red
- **`btn-warning`**: Yellow/Orange
- **`btn-info`**: Teal/Light Blue
